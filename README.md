# Elliyeen Get Started Page

## Option A: Existing Next.js project

1. Install lucide icons:

```bash
npm install lucide-react
```

2. Copy `page.tsx` into:

```bash
app/get-started/page.tsx
```

3. Run:

```bash
npm run dev
```

4. Visit:

```bash
http://localhost:3000/get-started
```

## Option B: New Next.js project

```bash
npx create-next-app@latest elliyeen-site --typescript --tailwind --app
cd elliyeen-site
npm install lucide-react
mkdir -p app/get-started
# copy page.tsx into app/get-started/page.tsx
npm run dev
```
