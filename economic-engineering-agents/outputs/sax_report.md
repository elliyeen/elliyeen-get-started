## Recommended First Experiment

**Experiment Name:** Private Wealth Clarity Report CTA Test

**Hypothesis:** If we replace the current website consultation CTA with a "Get Your Private Wealth Clarity Report" offer (gated PDF), then qualified consultations booked per week will increase by ≥25% within 6 weeks, because HNW prospects in the research phase prefer to evaluate the advisor's thinking before committing to a call — and the report demonstrates that thinking before any obligation is created.

**Control:** Current website homepage and service page with existing consultation CTA (whatever it currently reads).

**Variant:** Same pages with the primary CTA changed to: "Get Your Private Wealth Clarity Report →" with a 3-field form (Name, Email, Biggest financial challenge right now). The report is a 4–6 page PDF demonstrating SAX's framework and expertise. Auto-delivered via email. Follow-up sequence begins at download.

**Primary Metric:** Qualified consultations booked per week (defined as: prospect has ≥$500K investable assets, is not currently satisfied with existing advisor, and has a specific financial challenge SAX can address).

**Secondary Metrics:** Total report downloads, email open rate on follow-up sequence, close rate of report-sourced consultations vs. baseline.

**Success Threshold:** Qualified consultations ↑ ≥25% over 6-week period vs. prior 6-week baseline.

**Duration:** 6 weeks (need minimum 20 qualified consultations across the test period to draw conclusions).

**Decision Rule:**
- If qualified consultations ↑ ≥25%: implement report CTA as primary CTA permanently. Develop second report targeting a different HNW problem.
- If qualified consultations ↑ <25% but downloads are high: report is attracting traffic but follow-up is failing — test follow-up sequence next.
- If qualified consultations ↑ <25% and downloads are low: report topic or title is not resonating — test a different topic or headline.
- If total consultations increase but qualification rate drops: report is attracting wrong audience — tighten qualification language in the form.

**Risk Controls:**
- Existing clients and pipeline should not be affected — this test only changes the cold traffic entry point
- Maintain current direct contact methods (phone, email) during test period
- If consultation volume drops below a floor that hurts revenue, pause and revert to control

**Estimated Cost to Run:** Report design and writing ($500–$2,000 or internal), landing page update ($0–$500), email automation setup ($0–$200). Total: $500–$2,700 + 2–3 days of internal time.
